== Description ==
MACHER.one is a central WordPress Suite that allows you to manage backend modules like custom login design, AI media labeling, and other admin features — all from one dashboard.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the “Plugins” menu in WordPress.
3. Open “MACHER.one” from the admin menu to access settings and modules.

== Frequently Asked Questions ==

= Do I need MACHER.one to use sub-modules like AI Label? =
Yes. MACHER.one acts as the main framework. Extensions such as AI Label or Backend Login require it to be active.

= Is MACHER.one compatible with multilingual sites? =
Yes. It supports WordPress translation standards and currently includes English and German.

= Can I deactivate modules independently? =
Yes. Each module like "Backend Login" or "AI Media" can be enabled or disabled individually in the dashboard.
